### Mission: 우주와 지구를 잇는 데이터 파이프라인 설계자  
저는 대용량 데이터 엔지니어링과 항공우주 도메인 지식을 결합해, 뉴스페이스 시대의 데이터 병목을 해결하는 Space Data Systems Engineer입니다. 위성에서 수집되는 방대한 텔레메트리 데이터와 지구관측 영상이 단순한 숫자나 이미지에 머무르지 않고, 기후 위기 대응과 우주 환경의 지속가능성에 기여하는 실질적인 가치로 이어지도록 End-to-End 데이터 파이프라인을 설계합니다.

### Journey: 데이터 과학에서 글로벌 우주 산업으로
컴퓨터 과학과 데이터 분석을 기반으로 정보융합학을 전공하며, 데이터 기술을 우주 산업에 적용하겠다는 목표를 일찍부터 세웠습니다. 이후 UC San Diego 교환학생과 Calit2 Lab 연구원으로 참여하며 글로벌 연구 환경을 경험했습니다. 이 과정에서 드론 기반 다분광 센서 데이터를 활용한 실시간 지리참조 파이프라인을 개발하며, 노이즈가 많고 기준 데이터가 부족한 실제 공간 데이터를 다루는 역량을 키웠습니다.

### Expertise: 물리적 한계를 고려한 데이터 시스템 설계
우주 데이터 시스템은 제한된 통신 환경과 다운링크 대역폭이라는 물리적 제약을 항상 고려해야 합니다. Planet Labs에서 미션 데이터 엔지니어로 근무하며, 엣지 단에서의 텔레메트리 압축 로직을 개선해 데이터 전송 효율을 높였습니다. 또한 Kafka와 Apache Flink를 활용해 하루 240만 건 이상의 이벤트를 처리하는 실시간 스트리밍 아키텍처를 구축했습니다.
SIA와 SkyTruth 인턴십에서는 SAR와 광학 영상을 결합한 객체 탐지 모델을 개발하고, 이를 Docker와 Kubernetes 기반으로 배포하며 MLOps 전반을 경험했습니다.

### Vision: 오픈소스와 지속가능한 우주 환경
기술은 커뮤니티를 통해 완성된다고 믿습니다. NASA Space Apps Challenge에서 LEO 군집 위성의 충돌 경보 시스템을 개발해 Global Nominee로 선정되었고, Satpy 등 오픈소스 프로젝트에도 기여해왔습니다. 앞으로도 우주 데이터 시스템의 복잡한 문제를 해결하는 엔지니어로 성장하며, 데이터를 통해 지구와 우주를 더 깊이 이해하는 데 기여하고자 합니다.
